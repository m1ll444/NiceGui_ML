# NiceGui_ML
Interfaz Web con Python

## from-import
    Sirve para escribir menos código y hacer que sea más fácil de leer, ya que te permite llamar a las funciones o clases por su nombre directo.

## Def
    Sirve para crear "paquetes de instrucciones" que el programa ejecutará cada vez que el usuario presione un botón de la calculadora.

## ui.card
    Sirve para crear el cuerpo o "chasis" de la calculadora, agrupando todos los componentes dentro de un recuadro organizado y estético.

## .classes
    Sirve para aplicar el diseño a los componentes, controlando sus colores, tamaños y la separación entre ellos para que la interfaz sea visualmente atractiva.

## ui.label
    Sirve para representar cualquier tipo de mensaje o dato textual en la pantalla, ya sea un título fijo o un resultado que cambia según las acciones del usuario.

## ui.input
    Sirve para crear campos de texto donde el usuario introduce los datos numéricos necesarios para que la calculadora pueda realizar las operaciones.

## ui.button
    Sirve para crear los botones de la calculadora que, al ser pulsados, activan las operaciones matemáticas y muestran el resultado al usuario.

## on_click
    Sirve para vincular un botón con una función de Python, logrando que al pulsar el botón se ejecute la operación matemática correspondiente.

## ui.run
    Sirve para encender el motor de la aplicación y abrir la interfaz de la calculadora en el navegador para que el usuario pueda empezar a usarla.
